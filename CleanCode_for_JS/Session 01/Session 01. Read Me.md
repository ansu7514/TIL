## 2. Javascript 특성 파악하기
### 몽키 패치 (Monkey patch)
런타임 중인 프로그램의 내용이 변경되는 행동을 의미 (aka 안티 패턴)
### 특징
- 문법 → 자바와 유사
- 문자열, 배열, 정규 표현식 → 펄과 유사
- 함수 → 오크와 유사
- 클로저, 스코프 환경 → 스키마와 유사
- 프로토타입 → 셀프와 유사
- 이벤트 → 하이퍼토크와 유사
<br>
